# DESCF Website Rebuild

Institutional website rebuild for Deep Ecology and Snake Conservation Foundation (DESCF).

## Planned Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Sanity CMS
- GROQ
- Vercel deployment

## Project Goal

Build a CMS-driven, SEO-friendly, accessible, mobile-first institutional website for DESCF with newsroom, programme pages, resources, events, and governance content.

## Status

Planning phase. No production application code has been generated yet.

## Important Rule

Do not invent real partners, donors, reports, achievements, or statistics. All organizational claims must be verified before publication.
