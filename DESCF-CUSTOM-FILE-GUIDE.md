# DESCF customized file guide

Create or review files in this order:

1. `netlify.toml` — Netlify deploy settings
2. `admin/index.html` — Decap CMS admin entry
3. `admin/config.yml` — CMS collections and Git Gateway config
4. `assets/css/styles.css` — site styles
5. `assets/js/site.js` — shared utilities and layout helpers
6. `assets/js/home.js` — home page renderer
7. `assets/js/page.js` — generic page renderer
8. `assets/js/lists.js` — current work and resources list renderer
9. `index.html` — homepage shell
10. `about/index.html` — about page shell
11. `current-work/index.html` — current work shell
12. `strategic-priorities/index.html` — priorities page shell
13. `resources/index.html` — resources page shell
14. `partner-with-us/index.html` — partner page shell
15. `content/settings/site.json` — navigation and footer settings
16. `content/pages/home.json` — homepage content
17. `content/pages/about.json` — about content
18. `content/pages/strategic-priorities.json` — priorities content
19. `content/pages/partner-with-us.json` — partnership content
20. `content/lists/current-work.json` — current work cards
21. `content/lists/resources.json` — resources cards

This customized package already includes DESCF-specific content in files 15–21.
