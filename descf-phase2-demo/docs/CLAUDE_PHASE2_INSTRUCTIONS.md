# Claude Instructions for Continuing DESCF Phase 2

Use this project carefully. Do not generate the full final website in one response.

## Rules

1. Work in small phases.
2. Do not invent real DESCF achievements, partners, donors, reports, events, or statistics.
3. Keep the stack: Next.js App Router, TypeScript, Tailwind, Sanity, GROQ.
4. Avoid unnecessary libraries.
5. Keep components reusable but not overabstracted.
6. Use accessible semantic HTML.
7. Keep public pages working even when Sanity has no content.
8. Use placeholders only when clearly labelled.

## Next Best Development Tasks

1. Add image rendering from Sanity assets.
2. Add site settings fetch for footer/contact.
3. Add real content entry guide for DESCF editors.
4. Add sitemap and robots.
5. Add basic category filter page for newsroom.
6. Add Bengali route strategy only after content model is tested.

- payment gateway
- member dashboard

- multilingual route system
